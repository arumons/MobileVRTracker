# MobileVRTracker
ARKitを使ったモバイルVRポジショントラッキングを可能にしたスクリプトです。

# 使い方
#### 初期設定
UnityARKitPluginが別途必要です。<br>
https://bitbucket.org/Unity-Technologies/unity-arkit-plugin<br>
**※現在ARKitバージョン1.5が公開されていますが、当スクリプトは1.5に対応していません。旧バージョンをご利用ください。**
